Reusable HTML Components

1. Theory

1. Simple Components

Simple components have a fixed content. They can take attributes such as id or class and can be styled with external css.

Components can have their own functions.

You have not clicked the button yet

A component's attibutes can be tracked. If a value is given inside the component definition, that value overrides the values given manually.

Attributes can be used to customise a component content.

2. Templated Components

Components content can be customised by using templates and slots.

custom

Because of the use of shadow DOM, templated components content can't be styled with external css.

Slots of templated components with an inner styling can still be styled with external css.
